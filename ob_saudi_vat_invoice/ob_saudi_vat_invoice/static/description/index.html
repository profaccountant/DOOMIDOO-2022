ZATCA QR Code Specifications:

    Key Features
  • It is mandatory to generate and print QR code encoded in Base64 format with up to 500 characters that must contain the fields specified in the below table as per Annex (2) of the Controls,Requirements, Technical Specifications and Procedural Rules for Implementing the Provisions of the E-Invoicing Regulation.
  • QR code will be Print based on TLV QR Code.

Company Name, Company arabic name, Company Tax Details and additional Tax Detials as per the saudi Government regulations.
